## Tech Stack

---

- [TypeScript](https://www.typescriptlang.org/)

## How to Run?

---

Clone the repository

```bash
https://github.com/devlive-community/shadcn-ui-vue-admin
```

Install dependencies

```bash
npm install
```

Run

```bash
npm run dev
```
